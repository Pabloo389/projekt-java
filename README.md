# Projekt Java

## Opis

Ten projekt to przykładowa aplikacja Java zainicjowana w celu nauki Gita i GitHuba.

## Struktura katalogów

```
projekt-java/
├── src/         # kod źródłowy
├── bin/         # pliki binarne
├── .gitignore   # plik ignorujący
└── README.md    # dokumentacja projektu
```

## Uruchomienie

```bash
javac -d bin src/Main.java
java -cp bin Main
```

## Wymagania

- Java 11+
- Git
- Terminal
